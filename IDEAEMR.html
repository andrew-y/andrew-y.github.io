The IDEA-EMR Project

September 2015 - June 2016

Summary

The goal of this project was to replace paper-based emergency medical record systems in Kenya and Uganda, which were easily lost and inefficient for doctors. The project was developed on Android Studio and was eventually handed off to a Canadian non-profit organization called ICChange. ICChange used the project to study emergency room trends and eventually implemented the application in various hospitals in the region.

WHAT DID I WORK ON?

  • Programmed a large portion of the UI and front-end for the project using Java and XML on Android Studio
