Reflow Oven Controller

February 2017

Summary

The goal of this project was to convert a standard toaster oven into a reflow oven using simple electrical components. A thermocouple was used to measure the temperature within the oven, and an AT89LP52 microcontroller was used to control the system. The completed system was then used to solder components for an F38x microcontroller PCB.
